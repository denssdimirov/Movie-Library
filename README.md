# Movie Library - Личная кинотека

**Автор:** Иван Петров  
**GitHub:** [github.com/ivanpetrov/movie-library](https://github.com/ivanpetrov/movie-library)

## Описание

Movie Library - это графическое приложение для хранения и управления личной коллекцией фильмов. Программа позволяет добавлять фильмы, фильтровать их по жанру и году выпуска, а также сохранять данные в JSON файл.

## Функциональность

- ✅ Добавление фильмов с указанием названия, жанра, года и рейтинга
- ✅ Отображение всех фильмов в таблице
- ✅ Фильтрация по жанру и году выпуска
- ✅ Удаление фильмов
- ✅ Сохранение и загрузка данных в JSON формате
- ✅ Валидация ввода данных

## Требования

- Python 3.7 или выше
- Tkinter (встроен в стандартную библиотеку Python)

## Установка и запуск

python Movie Library.py
